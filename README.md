Introduction
------------
We try to accomplish **ONE** Front-end Using Modular CSS & HTML.
We do this by showing you our conclusions in **living documentation**.


Installation
------------

### Requirements
* [Node.js](https://nodejs.org/en/)
* [Gulp](http://gulpjs.com/)
* typescript
* tsd
* ts-loader
* webpack
* TODO 

### Get started
> npm install TODO
> tsc
> npm start

=> Go to localhost:8080/

### Develop
> npm run dev-build
> npm run dist-build
